# 扩展效果

# 阴影

说明：通过复制弹幕，微调距离来实现阴影效果

使用：文本对象中右键，添加效果中选择[阴影]

模仿M7配置：浓度设置100，扩散度选择0

可用参数：X，Y，浓度（透明度）

# 竖列文字

说明：通过选择对齐方式 垂直向上[左]，能在aviutl预览打竖文字，最后转换xml时会转换成高级弹幕换行\n

使用：文本对象对齐方式选择 垂直向上[左]

# 其他颜色描边

说明：通过弹幕叠层模仿使用其它颜色的描边效果

使用：文本对象中右键，添加效果中选择[描边]

颜色选择除RBD(0,0,0)以外的颜色，程序转换导出时会复制文字，微改坐标，通过叠层模仿描边。

可用参数：大小（px），模糊（透明度）

# 句子展开收缩

说明：将弹幕逐字分开，通过移动距离不同，模仿文本伸展收缩效果

使用：文本对象中右键，添加效果中选择[动画效果]，在[高级弹幕]下找到[字间扩展收缩]，文本对象勾选[独立每个文字]，调整中心X（为字体大小*字数/2），X缩放率选择直线运动，调整X缩放率。

可用参数：X缩放率，中心X

# 文字背景

说明：使用方块"█"作为文字的背景

使用：文本对象中右键，添加效果中选择[动画效果]，在[高级弹幕]下找到[文字背景]

模仿M7配置：余白设为0

可用参数：颜色，透明度

# 纯色视频背景

说明：使用大量"微软雅黑"字体的方块"█"覆盖视频来作为背景

使用：时间轴上右键，添加媒体物件，选择[图形]，图形类型选择纯色

可用参数：颜色

# 层级

说明：M7可以通过时间差来实现层级，出现时间在后会在上面。通过调整Z轴数值，转换程序会相应增加减少毫秒数

使用：文本对象中调整Z轴数值

PS：相同出现时间的弹幕，后发基本也会在上面，只是有多1毫秒的时间差能绝对确保层级关系

# 中间点强制分弹幕[配置]

说明：勾选中间点强制分弹幕，转换程序会自动对中间点切割，分开多条弹幕，效果相当于aviutl的分割

使用：勾选中间点强制分弹幕

# 延长结束时间50ms[配置]

说明：因为PC端高级弹幕显示时间比实际输入的生存时间少（少帧），该配置会在转换程序中让每条弹幕结束时间延迟50ms

使用：exo转xml配置中勾选延长结束时间50ms
