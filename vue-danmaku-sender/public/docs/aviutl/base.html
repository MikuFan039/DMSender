# 基本操作

# 弹幕（文本）

使用：右键→添加媒体物件→文本

仿真M7配置：选择 左对齐[上] ，点击 粗体B

# 位置移动

使用：文本对象中点击X或Y，选择直线运动

在视频中移动，绿框为开始位置，红框为结束位置

# 颜色

使用：文本对象中选择颜色

扩展：在文本中使用标签颜色。

仅支持最开头使用且只有一个颜色，用作代码颜色快速预览效果，省去颜色代码转RGB。

格式：<#ff0000>文本

# 字体大小

使用：文本对象中调整[大小]

说明：调整范围在10到127之间

# 字体

使用：文本对象中选择字体

# 文字描边

使用：文本对象中右键，添加效果中选择[描边]

仿真M7配置：大小设为1，模糊设为100

高级效果：使用其它颜色的描边。

颜色选择除RBD(0,0,0)以外的颜色，程序转换导出时会复制文字，微改坐标，通过叠层模仿描边。

# Z轴、Y轴翻转

使用：Z轴在文本对象中调整[大小]，不支持Y轴效果预览

# 生存时间

使用：通过在时间轴上对文本的长度拖动

# 透明度

使用：文本对象中调整[透明度]
