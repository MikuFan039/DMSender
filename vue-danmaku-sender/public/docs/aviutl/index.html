# M7在Aviutl简单使用

新建工程850*560,30fps 视频缩放率43.4% 新建媒体物件-文本 左对齐[上] 粗体B

ps：以上为个人笔记本14寸屏幕下根据B站屏幕大小做出的配置 15寸屏幕下配置为760*510,38.8%

在顶部[设置]中，点击扩展剪辑设置 在扩展剪辑栏中右键，添加媒体物件，选择文本，输入文本（弹幕）内容 在画面中拖动文本即可移动到相对应位置

在文本对象属性中点击X或Y，选择直线运动，在画面中拖动文本（绿色虚线框为起始位置，红色虚线框为结束位置），即可制作出移动效果

# [EX]通过浏览器插件，在线视频加载xml文件预览弹幕效果

插件：https://github.com/DeltaFlyerW/DF_DanmakuExtension

使用方法：在exo转换xml后，可以先通过该插件在视频上加载高级弹幕，预览弹幕效果再发送

先打开b站视频，复制 预览命令显示区域 中的命令，打开插件后台background.html，输入复制的命令，再刷新视频（确保要先打开视频，输入命令，再次刷新视频）。刷新视频后就可预览加载本地高级弹幕的效果。
